Uma das formas mais populares de medir o nível de usabilidade de um software é o SUS (System Usability Scale), portanto utilizamos da metodologia para testar o nivel de usabilidade da nossa aplicação. Sendo assim criamos um formulário para cada página afim de testar toda aplicação!
